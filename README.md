# React

# Start React with Vite

+ npm create vite
+ for proceed press (y)
+ Add project name
After that
+ using cd enter the project folder
+ then --> npm install (Create node_modules folder)
+ for run --> npm run dev or npx vite
+ for build --> npm run build or npx vite build (create dist folder)
